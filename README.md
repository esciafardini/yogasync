# yogasync

**The Problem**
Too many yoga studios with too many websites that are so annoying to navigate

**The Desired Outcome**
A command that can be run to print out a unified data structure representing yoga classes for the day
It would be nice to filter out teachers I do not want to deal with and show prices per class


### Terminal Commands

Run the main function
- clj -X:run                                                                                                                                                                   ~/Projects/yogasync

Start a cider repl
- clj -M:repl
